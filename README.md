Juego: Adivina el Número

Este proyecto consiste en un juego sencillo desarrollado con HTML, CSS y JavaScript. El objetivo es que el usuario adivine un número aleatorio entre 1 y 100 en un máximo de 10 intentos.

Después de cada intento, el sistema indica si el número ingresado es mayor o menor que el número secreto. Además, se muestra el historial de números utilizados y la cantidad de intentos restantes.

Funcionamiento

Se genera un número aleatorio entre 1 y 100.

El jugador dispone de 10 intentos.

En cada intento:

Se valida que el número esté dentro del rango permitido.

Se indica si el número es mayor o menor que el valor secreto.

Se actualiza la lista de números ingresados.

Se muestra el número de intentos restantes.

El juego finaliza cuando el jugador acierta o cuando se agotan los intentos.

Al terminar, se ofrece la opción de reiniciar el juego.

Tecnologías utilizadas

HTML5 para la estructura.

CSS3 para el diseño y estilos.

JavaScript para la lógica del juego y la interacción con el DOM.
